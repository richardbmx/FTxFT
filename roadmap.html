Road Map

Junio 2019
Diseño de Sitio Web

Desarrollar y generar un sitio web que permita a los especialistas del área de la rehabilitación física, registrarse y mantener un usuario para ofrecer sus servicios a domicilio, con un sistema de calificación y clasificado por campo de especialidad y región .

Julio 2019
FTS Token ERC-20

Generar el token FTS basado en la tecnología Blockchaing de Ethereum con el protocolo ERC-20 para brindar seguridad a nuestros usuarios.

Diciembre 2019
Venta privada

Iniciar una venta privada del 10% del total de tokens para los primeros inversionistas interesados en desarrollar el proyecto.

Marzo 2020
Venta publica

Venta del 25% de los Tokens para el público en general, que podrán ser utilizados en el sitio web para contratar servicios de rehabilitación .

Junio 2020
Desarrollo de la APP

Creación de una APP para dispositivos iOs y Android que cumpla con las funciones del sitio web para garantizar usabilidad de todo el protocolo.
